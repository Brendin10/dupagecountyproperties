# Bandland

A **Unity 2D** rhythm gig game. Open **this folder** (the repo root) in Unity Hub.

## Open in Unity Hub

1. **Unity Hub → Open → Add project from disk**
2. Select **this folder** — the one that contains `Assets`, `Packages`, and `ProjectSettings`
3. Do **not** open a parent folder or a subfolder like `bandland/` (web prototype)

If Unity says "No Unity project found", you selected the wrong directory. The correct folder has:

```
Bandland/          ← open THIS folder
  Assets/
  Packages/
  ProjectSettings/
  README.md
```

## First-Time Setup

1. Use **Unity 2022.3 LTS** or newer
2. Import **TextMeshPro Essentials** when prompted
3. Menu: **Bandland → Setup Project (Run Once)**
4. Open `Assets/Bandland/Scenes/CharacterSelect.unity`
5. Press **Play**

## Gameplay

- Pick **Benny** or **Lizzy**
- Play gigs on the **beat lane** — tap when notes hit the gold line
- **Miss** = lose star power; **combo** = louder crowd
- Unlock venues as your Star Meter grows
